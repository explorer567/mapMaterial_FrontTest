如何使用开发者工具查看resize事件

1

打开浏览器开发者工具 (F12 或 Ctrl+Shift+I)

2

转到 "Performance" 或 "Performance" 标签

3

点击录制按钮，然后调整浏览器窗口大小

4

停止录制，查看分析结果中的 "Event" 部分

查找 "resize" 或 "ResizeObserver" 事件

5

替代方法：在 "Elements" 标签中，选择 <body> 元素

在 "Event Listeners" 选项卡中查找 "resize" 事件

代码示例：添加resize事件监听器

let count = 0;
window.addEventListener('resize', () => {
  count++;
  console.log(`Resize event triggered: ${count} times`);
});
